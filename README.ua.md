# Task Board (Vanilla JS)

Проста та функціональна канбан-дошка для керування завданнями, написана на чистому JavaScript. Проєкт створено для закріплення базових навичок роботи з DOM та сховищем браузера.

## 🚀 Стек технологій
* **Frontend:** JavaScript (ES6+), HTML5, CSS3
* **Storage:** LocalStorage (для збереження даних)

## 💡 Ключовий функціонал
* Створення, редагування та видалення завдань.
* Переміщення карток між колонками.
* Автоматичне збереження всіх змін у браузері (дані не зникають після перезавантаження сторінки).

## 🛠️ Як запустити локально
1. Склонуйте репозиторій: `git clone https://github.com/dino004/task-board.git`
2. Відкрийте файл `index.html` у будь-якому браузері (або використовуйте Live Server у VS Code).

## 📈 Плани з розвитку
* Переписати додаток на **React** для практики компонентного підходу.
* Створити повноцінний бекенд (Node.js/Express) та базу даних для зберігання завдань у хмарі.
